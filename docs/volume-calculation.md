# Оценка объёма БД

## Заказы

По заданию: `Минимум 1 год хранения для заказов и платежей.`

Верхняя оценка хранения при постоянной нагрузке:

Заказы/год:
```
100 (rps) × 60 × 60 × 24 × 365 ≈ 3.15 × 10^9 
```

Объём: 
```
3.15 × 10^9 × 1КБ ≈ 3.15ТБ
```

## Остальное

Считаем, что офферы и платежи в объеме имеют примерно 1:1 к заказу

## Итого (грубо по верхней оценке)

Порядка 9ТБ/год, с учетом компрессии, агресивных чисток - сильно меньше
